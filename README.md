# Техническое задание 

Клиент-серверная система численного интегрирования функции **1/ln(x)**.

Сервер принимает от оператора параметры интегрирования, распределяет задачу между подключёнными клиентами, клиенты параллельно вычисляют свои подотрезки на всех ядрах CPU и возвращают результаты. Сервер суммирует их и выводит итог.

---

## Зависимости

| Библиотека | Версия | Назначение |
|---|---|---|
| CMake | ≥ 3.16 | Сборка |
| C++ compiler | C++17 | GCC ≥ 9 / Clang ≥ 9 / MSVC 2019+ |
| Boost.Asio | ≥ 1.74 | Асинхронный TCP |
| Protobuf | ≥ 3.21 | Сериализация сообщений |

### Linux (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install cmake g++ libboost-all-dev libprotobuf-dev protobuf-compiler
```

### macOS

```bash
brew install cmake boost protobuf
```

### Windows

Рекомендуется использовать [vcpkg](https://github.com/microsoft/vcpkg):

```powershell
vcpkg install boost protobuf
```

## Сборка

```bash
# Клонировать репозиторий
git clone https://github.com/griggg/Razrabotchik_c
cd Razrabotchik_c

# Сконфигурировать
cmake -B build -DCMAKE_BUILD_TYPE=Release

# Собрать
cmake --build build --parallel
```

После сборки в папке `build/` появятся два исполняемых файла:
- `server` — серверное приложение
- `client` — клиентское приложение

## Запуск

> Сервер и клиент(ы) можно запускать на одной машине или на разных хостах в одной сети.

### 1. Запустить сервер

```bash
./build/server
```

Сервер начнёт слушать входящие подключения на порту **9000**.

### 2. Запустить один или несколько клиентов

В отдельных терминалах (или на других машинах):

```bash
./build/client
```

При успешном подключении клиент выведет:

```
Connected!
```

### 3. Отправить задачу интегрирования

В терминале сервера введите команду:

```
makeTask <left> <right> <step>
```

| Параметр | Описание |
|---|---|
| `left` | Нижний предел интегрирования (должен быть **> 1.0**, т.к. ln(1) = 0) |
| `right` | Верхний предел интегрирования |
| `step` | Шаг интегрирования (например, `0.001`) |

**Пример** — интеграл 1/ln(x) на отрезке [2, 10] с шагом 0.001:

```
makeTask 2 10 0.001
```

Пример вывода сервера:

```
Клиент добавлен, всего: 1
Клиент добавлен, всего: 2
Клиент добавлен, всего: 3
makeTask 2 10 0.00001
Клиент 0 получает: [2, 4.66667]
Клиент 1 получает: [4.66667, 7.33333]
Клиент 2 получает: [7.33333, 10]
Получен результат: 1.2393 (1/3)
Получен результат: 1.50339 (2/3)
Получен результат: 2.37784 (3/3)
=== ИТОГ: 5.12054 ===
```

## Принцип работы

```
Пользователь вводит: makeTask 2 10 0.001
        │
	▼
    [ Сервер ]
  делит [2,10] на N частей по числу клиентов
        │
   ┌────┴────┐
   ▼         ▼
[Клиент 1] [Клиент 2]   (каждый делит свой подотрезок
  делит      делит        ещё на число ядер CPU
  на потоки  на потоки    и считает параллельно)
   │         │
   └────┬────┘
        ▼
    [ Сервер ]
  суммирует результаты
  выводит итог
```
